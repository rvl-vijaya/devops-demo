\# DevOps Demo Project 🚀



A \*\*containerized Node.js web application\*\* demonstrating DevOps best practices: Docker, Docker Compose, CI/CD pipelines with GitHub Actions.



\---



\## 🛠 Technologies Used



\- \*\*Docker\*\* – Containerization

\- \*\*Docker Compose\*\* – Multi-container orchestration

\- \*\*Node.js\*\* – Web application

\- \*\*Git\*\* – Version control

\- \*\*GitHub Actions\*\* – Continuous Integration (CI)

\- \*\*NGINX\*\* – Web server (optional)

\- \*\*PowerShell / Terminal\*\* – Command line usage
